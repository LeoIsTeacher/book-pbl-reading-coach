# 默认规则

读者默认是已经接触主题、但尚未形成体系的普通学习者。一本书先做章节型路径，再在高手阶段组合多本书。任务以顺序编号而非日期组织，建议每个小项目约一周、总投入约十小时，但允许读者自行安排进度。

正文采用科普书式完整段落，中文自然段通常为200–500字。案例背景约800字或更长，必须包含人物、数据、限制、冲突和待决问题。每个任务明确标注版本、章节范围与章节标题，要求读者分析、批判、说服、重写或设计方案，并以社群提交和AI反馈收束。

作者与时代背景由AI提供事实脚手架，包含可核验数据与明确标注的虚构情境；读者负责判断长期原则、时代经验和现实迁移。涉及隐私时使用100单位匿名模型，不收集真实收入、负债、健康或身份数据。
