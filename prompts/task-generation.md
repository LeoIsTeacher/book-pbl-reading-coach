# 单个任务生成提示词

根据已批准的任务地图，只生成当前任务。开头用一至两个完整段落说明为什么研究这个问题，随后写一个具有真实感的小型PBL案例。案例包含人物、具体数字、家庭或职业责任、互相冲突的选择、信息缺口和无法轻易回避的决策；虚构案例需说明其为教学情境，不伪装成真实新闻。

用完整段落写明本任务阅读范围，包括第几章到第几章和章节标题，并说明带着什么问题阅读。读后任务要求读者回到案例，完成分析、批判、说服、改写或方案设计，并说明判断依据、现实边界与验证步骤。避免密集问题列表和机械填空。

结尾说明读者掌握的核心问题解决方法，邀请其把作业交给社群主理人或通知主理人前往GET大脑查看。说明主理人与AI将评价关键变量、书中观点使用、现实限制和最有价值的思考，并安排匿名同伴讨论。生成后停止，等待明确的`pass`，未经批准不得发布。
