# 任务0批改指南：阅读背景与学习期待

## 定位与阅读范围

阅读序言“这就是你所需要的”，暂不进入正文。任务0是诊断性基线：记录成员当前如何理解财富、最信任哪种积累方式、最想解决什么问题，以及完成全书后希望发生什么变化。不要急于纠正，更不要提前灌输后续章节的答案。

## 核心观察

观察财富定义是否具体。账户金额、稳定收入、资产、时间、选择权、安全感和家庭保障都可以成立，重点是成员能否说明为什么重要，以及能否转化为可观察状态。观察其信任工资、储蓄、房产、股票、技能或生意的理由来自亲身经验、家庭环境，还是未经验证的成功故事。把“想学投资”“想财务自由”继续追问为具体场景，例如不知道怎样分配收入、无法判断资产、市场下跌时恐慌或学了很多却不敢开始。

高质量回答不一定术语多，而是能够说明：我目前为什么这样想，这种想法来自什么，它在哪些条件下可能成立，我还有什么不确定。成员可以暂时不信任任何投资方式，这同样是有效起点。

## 常见误区与批改边界

警惕直接复述序言、用“资产让钱工作”等口号代替自己的观点；用宏大目标回避具体问题；把一次成功或失败推广为普遍规律；为了显得正确而隐藏不确定。不要打分，不因未使用资产、负债、现金流等术语而判断不足，不把自住房或工资等后续争论提前塞入反馈，也不与其他成员比较。

## 快速反馈

先指出最清晰、最具个人经验的起点；再选择财富定义、信任依据、当前问题或学习目标中的一个模糊变量；最后提出一个能把起点具体化的追问。可用追问包括：你说的“有钱”具体是什么生活状态？这种积累方式曾在什么情况下保护你，又可能何时失效？完成本书后，你希望面对哪种决定时作出不同反应？

## 成长档案

只记录当前财富定义、最信任方式及理由、最想解决的问题、希望发生的变化、本次最有价值思考和下一次观察项。所有能力或问题均标记为初始迹象或暂不判断，不能记录稳定能力和反复盲点。
